# ML Notebooks

A collection of machine learning and data science notebooks covering a range of techniques and real-world datasets.

## Notebooks

| Notebook | Description |
|----------|-------------|
| `Decision Tree Classifier for Breast Cancer Data.ipynb` | Classification using decision trees on the Wisconsin Breast Cancer dataset |
| `Naive Bayes.ipynb` | Probabilistic classification using Naive Bayes |
| `K Means.ipynb` | Unsupervised clustering with K-Means |
| `K fold experiments.ipynb` | Cross-validation experiments comparing model performance |
| `Regression for Male vs Female Life Expectancy by London Borough.ipynb` | Linear regression analysis of ONS life expectancy data across London boroughs |
| `Graph network analysis.ipynb` | Network graph analysis and visualisation |
| `Pacman Reinforcement Learning.ipynb` | Reinforcement learning agent for the Pacman environment |
| `Regression test.ipynb` | Regression experimentation and model tuning |

## Requirements

```bash
pip install numpy pandas scikit-learn matplotlib seaborn networkx jupyter
```
